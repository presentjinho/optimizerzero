# OptimizerZero 전략

## 목표

방문자가 언제든 다시 쓸 수 있는 압축 도구.

- 설치 없이 웹에서 바로 사용
- 한 번 접속하면 PWA 캐시로 재방문/오프라인 사용 가능
- 원본을 서버로 보내지 않는 로컬 처리
- 사용 목적에 따라 압축 방식을 추천

## 차별점

일반 압축기는 보통 “압축률” 중심입니다. OptimizerZero는 “사용자가 감수할 손실과 목적” 중심입니다.

- 보관용: 원본 보존, 무손실 중심
- 공유용: 적당한 용량과 품질 균형
- 메신저용: 용량 우선, 품질 확인 권장
- 이메일용: 첨부 제한 중심
- 품질 우선: 거의 티 안 나는 절감만 허용

## 제품 메시지

“얼마나 줄일지, 얼마나 손실을 허용할지 먼저 고르는 압축기.”

보조 메시지:

- 서버 업로드 없음
- 원본 보존
- 결과 검증
- 손실 허용도 선택
- 목표 용량/품질 직접 조절
- 결과 리포트 제공

## 최적 배포 방안

1. Netlify에 `web/` 배포
2. PWA로 설치 가능하게 유지
3. 첫 화면은 작업 도구, 설명 페이지 아님
4. X 공유는 “파일 안 올리는 압축기”보다 “목적별 압축 추천”을 강조
5. 데스크톱 ZIP은 대용량/폴더용 고급 버전으로 제공
