# X 공유 초안

## 짧게

OptimizerZero 만들었습니다.

파일을 서버에 올리지 않고 ZIP/CBZ/EPUB/Office/이미지를 줄이는 로컬 압축 도구입니다.
원본 보존, dry-run, 검증, 손실 허용도/품질/목표 용량 조절을 기본으로 넣었습니다.

## 기능 강조

압축 도구에서 제일 불안한 건 “내 원본이 망가질까?”라서 OptimizerZero는 원본을 기본 보존합니다.

- 손실 없음/낮음/중간/높음 선택
- JPEG/WEBP 품질 직접 지정
- 목표 용량 지정
- 너무 작은 절감은 스킵
- 결과 검증 후 저장

## 웹 버전

설치 싫은 사람용 Web Lite도 있습니다.
파일은 브라우저 안에서 처리되고 서버로 업로드되지 않습니다.
